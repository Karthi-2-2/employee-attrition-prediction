# Employee Attrition Prediction

## 📌 Overview
This project predicts employee attrition using machine learning techniques.

## 🎯 Objective
- Analyze employee data
- Identify key attrition factors
- Build predictive models

## ⚙️ Technologies
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

## 🤖 Models Used
- Logistic Regression
- Random Forest

## 📊 Evaluation
- Accuracy
- Precision & Recall
- ROC-AUC Score

## 🚀 How to Run
pip install -r requirements.txt
python attrition_model.py
